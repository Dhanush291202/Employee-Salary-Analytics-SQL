Recruitment Analytics Dashboard (Power BI)

This project focuses on analyzing recruitment performance through a Power BI dashboard.
The idea was to track key metrics from candidate sourcing to onboarding and understand how data can make hiring more efficient.

Objective

To visualize and measure:

Candidate pipeline (Sourced → Interviewed → Offered → Joined)

Time taken for each hiring stage

Recruiter performance and contribution

Effectiveness of different hiring sources

Tools Used

Microsoft Excel – for creating and cleaning the dataset

Power BI – for building the visuals and dashboard

Insights Gained

LinkedIn and referrals showed higher joining conversion rates

Average time-to-hire was around 8–10 days

Offer acceptance stood at nearly 70 percent

Identified areas where the process could be streamlined

Key Takeaways

Creating this dashboard helped me understand:

The impact of analytics on recruitment decisions

How visualization makes data easier to explain

The importance of tracking HR metrics continuously
